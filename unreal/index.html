Unreal-Android View on GitHub

Android port of Unreal (1998)

OUYA + Android 8+ WiFi Multiplayer Controller Friendly

An Unreal port for Android and OUYA, with SDL2 input, OpenGL ES rendering, controller support (no onscreen buttons) and local WiFi multiplayer support.

Download at GitHub

Highlights

  • Support for newer devices in native resolution
  • Dedicated OUYA build with lower resolution
  • Local WiFi multiplayer and botgames
  • Controller-friendly gameplay

Requirements

  • Android 8.0+ or Android 4.x for OUYA
  • OpenGL ES 2.0 compatible device
  • Original Unreal v200 or v205 demo data
  • Game controller necessary

Android Port Changes

  • Android Studio / Gradle / NDK project setup
  • CMake integration for native UE1 sources
  • SDL2 window, input and lifecycle handling
  • OpenGL ES rendering adjustments
