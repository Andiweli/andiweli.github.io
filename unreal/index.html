Unreal-Android

Android port of Unreal / Unreal v200 (1998)

Work in Progress Android 8+ OUYA 960x540 WiFi Multiplayer

A community-driven Unreal Engine 1 port for Android, including a dedicated low-resolution OUYA build for smoother performance on Tegra 3 hardware.

GitHub Repository Port Highlights

Highlights

  • Android 8+ support on newer devices
  • OUYA build optimized for 960x540
  • Local WiFi multiplayer and botgames
  • Controller-friendly gameplay

Requirements

  • Android 8.0+ or Android 4.x for OUYA
  • OpenGL ES 2.0 compatible device
  • Original Unreal v200 or v205 demo data
  • Game controller recommended

Android Port Changes

  • Android Studio / Gradle / NDK setup
  • CMake integration for native UE1 sources
  • SDL2 window, input and lifecycle handling
  • OpenGL ES rendering adjustments
