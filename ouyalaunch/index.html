Fast OUYA-styled launcher for the OUYA micro-console

Android 4.1.x Favorites + Genres Controller Only

OUYAlaunch is a custom home screen replacement built for the original OUYA console. It keeps the familiar dark cinematic feel, adds favorites and genre tabs, speeds up browsing with cached icons and stays fully focused on fast, controller-only navigation.

Download at GitHub

Highlights

  • OUYA-styled clean interface
  • Favorites row and six genre tabs
  • Fast icon caching for smoother browsing
  • Controller-only navigation
  • Automatic app discovery

Requirements

  • OUYA console or Android 4.1.x device
  • Game controller / OUYA controller
  • APK installation from GitHub releases
  • Apps with “ouya_icon.png” supported best

How it works

  • Scans installed apps on first launch
  • Looks for “ouya_icon.png” resources
  • Caches matching icons for fast grid drawing
  • New apps can be moved between genre tabs
  • Favorites and rescan are available anytime
OUYAlaunch is an open-source side project by @andiweli to give the OUYA a modern, focused launcher while staying true to the original console look.
